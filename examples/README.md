# Example Resume

`example-resume.json` is a sample resume for Alex Chen — a full-stack engineer.

This file is all fake data (name, email, phone, companies are fictional). Use it to:

- **Builder**: open the Layout Builder to see how sections, margins, fonts, and compact mode render
- **Tailor**: start an AI Tailor session to see how the agent suggests improvements against a JD
- **Interview**: pick an interviewer and run a mock interview based on this resume

## Try it

1. Start the backend and frontend (see root README)
2. On the dashboard, click **Import Resume** and drop `example-resume.json`
3. Or copy-paste the JSON content directly into the editor

## What's inside

| Section | Content |
|---------|---------|
| Personal Info | Name, title, email, phone, location, GitHub, LinkedIn |
| Summary | 2-line career overview |
| Work Experience | 3 roles (Senior SWE, SWE, Junior Dev) with bullets |
| Education | University of Washington (MS), UC Berkeley (BS) |
| Projects | 2 personal projects with descriptions |
| Skills | Languages, frameworks, cloud, databases |
| Languages | English (Native), Mandarin (Fluent) |
